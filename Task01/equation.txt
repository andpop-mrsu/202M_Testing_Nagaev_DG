# Тестовые случаи для проверки функции solve_quadratic(a, b, c)

1. a=1, b=-3, c=2
   Ожидаемый результат: [1.0, 2.0]
   Описание: Обычный случай с двумя различными вещественными корнями.

2. a=1, b=2, c=1
   Ожидаемый результат: [-1.0]
   Описание: Случай с одним вещественным корнем (дискриминант равен нулю).

3. a=1, b=1, c=1
   Ожидаемый результат: "Нет вещественных корней"
   Описание: Случай, когда нет вещественных корней (дискриминант отрицательный).

4. a=0, b=2, c=-4
   Ожидаемый результат: [2.0]
   Описание: Линейное уравнение, преобразованное из квадратного при a=0.

5. a=0, b=0, c=5
   Ожидаемый результат: "Нет решений"
   Описание: Случай, когда уравнение не имеет решений (противоречие).

6. a=0, b=0, c=0
   Ожидаемый результат: "Бесконечно много решений"
   Описание: Случай, когда любое значение x является решением (тождество).

7. a=-1, b=0, c=4
   Ожидаемый результат: [-2.0, 2.0]
   Описание: Обычный случай с двумя различными вещественными корнями, где a < 0.

8. a=1, b=-4, c=4
   Ожидаемый результат: [2.0]
   Описание: Случай с одним вещественным корнем (дискриминант равен нулю).
